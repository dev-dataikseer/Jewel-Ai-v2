7.2 Tech Stack (Updated)
-------------------------
Frontend    : Next.js 14 (React), Tailwind CSS, Shadcn/UI, Lucide React
Backend     : Node.js (Express.js), TypeScript
Database    : PostgreSQL (Perfect relational DB for jewelry metadata)
ORM         : Prisma
Job Queue   : BullMQ + Redis (Handles async AI processing)
File Storage: Hostinger Local Storage / S3-Compatible Storage
Hosting     : Hostinger VPS (Ubuntu)
Deployment  : Docker + Docker Compose

7.3 Deployment Strategy — Hostinger
------------------------------------
- The application is containerized using Docker.
- Deployment is on a Hostinger VPS (CyberPanel or plain Ubuntu).
- Nginx acts as a reverse proxy with SSL via Let's Encrypt.
- PostgreSQL and Redis run as Docker services.

7.4 Mobile Optimization
------------------------
- Progressive Web App (PWA) features for mobile home screen installation.
- Responsive design using Tailwind's mobile-first breakpoints.
- Touch-friendly controls and optimized image loading.
- High performance (Lighthouse score target > 90).

================================================================================
9. FUNCTIONAL & NON-FUNCTIONAL REQUIREMENTS
================================================================================

9.1 Functional Requirements
---------------------------
- F1: User must be able to upload jewelry images (up to 20MB).
- F2: 1-Click Auto-Enhance for background removal and lighting fix.
- F3: Structured selection of Jewelry parameters (Metal, Gemstone, etc.).
- F4: Prompt-based editing with negative prompt safety.
- F5: Virtual try-on with auto-landmark detection and manual adjustment.
- F6: Credit-based payment system with Stripe.
- F7: Shareable links and WhatsApp integration.

9.2 Non-Functional Requirements
-------------------------------
- NF1: Security - JWT authentication and encrypted storage for API keys.
- NF2: Performance - Processing time < 60s for most jobs.
- NF3: Reliability - 99.9% uptime for core database and API.
- NF4: Scalability - Queue system to handle peak loads.
- NF5: Usability - Mobile-optimized interface with < 100ms interaction latency.
